Linear Interpolator, Single Variable

Instructions for using the Linear Interpolator:

1. Enter the known variable pairs (both x & y) into the shaded boxes that surround the white box.

2. Enter the middle point desired coordinate (either x or y), and leave the unknown value blank to be interpolated.

3. Click interpolate. The given value should be within the range of the other variables if your input is valid.
