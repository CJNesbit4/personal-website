Liquid-Liquid Equilibrium Image Transformer

Instructions for using the Liquid-Liquid Equilibrium Image Transformer:

1. Generate the Equilateral LLE curve in ChemCAD. Create a screenshot image of the diagram.

2. Crop the saved image so that the triangle points contact the sides of the image. See the example image at the bottom of this page if this is unclear.

3. Upload the cropped image to this page. If the preview is satisfactory, click "Download" to save the modified image.
